Code I wrote for the assignment for the course "model checking and parity games".

On Ubuntu, the "build.sh" script can be run to install the necessary tools for building the project and will build an executable.
This executable may be found in "dist-newstyle\build\x86_64-linux\ghc-9.10.3\ParityGames-0.1.0.0\x\ParityGames\build\ParityGames\ParityGames".
GHCUP will prompt some questions about the installation. For all of these the default is fine.

The executable may be run with the --help option for an overview on how to use it.
